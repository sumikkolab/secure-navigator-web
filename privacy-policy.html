プライバシーポリシー

最終更新: 2026年3月25日

1. データの収集

SecureNavigator は、利用者の個人情報を収集、送信、共有しません。本製品は完全にオフラインで動作します。

2. ネットワーク通信

本製品は外部ネットワークへの通信を一切行いません。診断データ、回答内容、レポートはすべて利用者のローカル環境内で処理されます。

例外: Microsoft Store からのインストール・更新、および Store ライセンス状態の確認には、Microsoft のネットワーク通信が発生します。これは本製品ではなく OS / Store の機能です。

3. ローカルデータの保存

本製品は以下のデータを利用者のローカルデバイスに保存します。

  • 診断セッション（回答内容、評価結果）
  • アプリケーション設定（言語、ライセンス状態）

保存場所: %LocalAppData%\SecureNavigator\

これらのデータは外部に送信されることはありません。

4. 第三者との共有

本製品は利用者データを第三者と共有しません。分析、広告、トラッキングのためのデータ収集は行っていません。

5. サポート対応時のデータ取り扱い

サポートのお問い合わせに際して、利用者が任意でスクリーンショットやログを提供された場合、これらは問題解決の目的にのみ使用し、解決後に削除します。

6. ポリシーの変更

本ポリシーは予告なく変更されることがあります。重要な変更がある場合は、本ページを更新してお知らせします。

7. お問い合わせ

プライバシーに関するご質問は、サポートページからお問い合わせください。

Product Info

Privacy Policy

Last updated: March 25, 2026

1. Data Collection

SecureNavigator does not collect, transmit, or share any personal information. The product operates entirely offline.

2. Network Communication

This product does not perform any external network communication. All assessment data, responses, and reports are processed exclusively within the user's local environment.

Exception: Installation and updates via Microsoft Store, and Store license verification, involve Microsoft's network communication. This is a function of the OS / Store, not this product.

3. Local Data Storage

This product stores the following data on the user's local device:

  • Assessment sessions (responses, evaluation results)
  • Application settings (language, license state)

Storage location: %LocalAppData%\SecureNavigator\

This data is never transmitted externally.

4. Third-Party Sharing

This product does not share user data with third parties. No data collection for analytics, advertising, or tracking is performed.

5. Data Handling During Support

If you voluntarily provide screenshots or logs for support purposes, these will be used solely for troubleshooting and deleted after resolution.

6. Changes to This Policy

This policy may be updated without prior notice. Significant changes will be communicated by updating this page.

7. Contact

For privacy-related inquiries, please visit our Support page.
